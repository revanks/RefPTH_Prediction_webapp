RefPTH Prediction

RefPTH Prediction is AI based Real-Time
Refrigerant Pipe Thermo-Hydraulics Evaluation Tool

Motivation
Two-phase pressure drop prediction of refrigerant fluids is very essential in many cooling applications.
In the past many decade's thermal system design engineers solely relied on many empirical and semi-empirical pressure drop models which are developed for specific fluids and a narrow range of operating conditions, this makes prediction accuracy poorer.
The RefPTH Prediction tool evaluates two-phase frictional pressure drop and flow regime of refrigerant pipes in real-time based on Machine Learning models which are built on experimentally measured data points resulting in accurate predictions than any other conventionally available methods.
About the tool

The RefPTH Prediction tool offers computationally inexpenxive real-time prediction of following use cases
1. Two-Phase Pressure Drop Prediction in Horizontal Straight Pipe
 2. Two-Phase Flow Regime Prediction in Horizontal Straight Pipe
 3. Two-Phase Pressure Drop Prediction in Return Bend Pipe
